Angular Interpolation / String Interpolation

String Interpolation uses template expressions in double curly braces to display data from the component, the special syntax, also known as moustache syntax.The contains JavaScript expression which can be run by Angular and the output will be inserted into the HTML.

{{"This content is" + interpolationContent + "optained through Interpolation."}}

Say if we put 5 + 5 in the double curly brackets 10 will be inserted into the HTML

{{5+5}}

Limitations
  • assignemts are not allowed inside curly braces,it gives parse Error
  • It doesent provide acces to global variable i.e template not aware of global Javascript variables like (window.location.href). Insted of that we have to define it in the class and bind it to the Template an it will provide expected result. Example:

    {{siteUrl}}

  • Interpolation works with Sring only thats why we need property binding as values can be boolean or number
